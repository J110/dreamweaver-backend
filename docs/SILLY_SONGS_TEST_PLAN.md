# Dream Valley — Silly Songs Test
## 6 Songs via ACE-Step — Experiment Before Spec

---

## Goal

Generate 2 funny songs per age group (6 total) using ACE-Step. Evaluate whether the output is catchy, funny, and child-appropriate enough to become a Before Bed content type. No full spec until we hear the results.

---

## ACE-Step Basics

ACE-Step takes:
- **Style/tags prompt:** genre, mood, instruments, tempo
- **Lyrics:** with structure tags `[verse]`, `[chorus]`, `[bridge]`
- **Duration:** up to 240 seconds (we want 60-90 seconds)

## ACE-Step Generation

```bash
# Generate each song
for song in not_a_rock sneezy_bear perfectly_normal the_plan im_fine the_explanation; do
  python3 scripts/generate_song.py \
    --style-prompt "$(cat prompts/${song}_style.txt)" \
    --lyrics "$(cat prompts/${song}_lyrics.txt)" \
    --duration 80 \
    --output "output/songs/${song}.wav"
done
```

If you don't have a `generate_song.py` wrapper yet, the raw ACE-Step call:

```python
import replicate

output = replicate.run(
    "lucataco/ace-step",
    input={
        "tags": style_prompt,        # the style/genre description
        "lyrics": lyrics_text,       # the structured lyrics
        "duration": 80,              # target seconds
        "num_samples": 2,            # generate 2 variants, pick best
    }
)
```

Generate **2 variants per song** (different seeds). Pick the better one. ACE-Step output quality varies significantly between seeds — the "gacha" nature of the model means some generations are great and some are mediocre.

## What to Evaluate

Listen to all 6 and score each on:

| Criteria | What to Listen For | Pass/Fail Threshold |
|---|---|---|
| **Chorus catchiness** | Can you hum the chorus after one listen? Would a child sing it? | If you can't remember it after 2 listens, fail |
| **Vocal clarity** | Are the words understandable? Can you follow the lyrics? | If more than 20% of words are garbled, fail |
| **Comedy timing** | Do the funny moments land? Does "Oh no... it broke" have a beat? | If the punchlines are buried in the melody, fail |
| **Age appropriateness** | Does the musical style match the age? 2-5 should feel like a nursery rhyme, 9-12 should feel like indie pop | If a 10-year-old would cringe at song 5, fail |
| **Production quality** | Does it sound like a real song or AI slop? | If it sounds broken or glitchy, fail |
| **Singability** | Could a child actually sing along? Are the words rhythmically placed? | If the lyrics fight the melody, fail |
| **Length** | Is 60-90 seconds enough for the song to feel complete? | If it feels cut off or too long, note the ideal length |

### The One Test That Matters

Play "Not a Rock" for your daughter. Does she sing "he's not a rock, he's a croc" afterward? If yes, this works. If she doesn't react, the melody isn't catchy enough — try different ACE-Step seeds or switch to MiniMax Music 1.5.
